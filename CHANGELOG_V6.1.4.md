# V6.1.4 — 地方公職資料解析修正版

## 修正重點

- 修正內政部「地方公職人員資訊專區」人物卡片解析。
- 不再只依賴「詳細資訊」文字的固定相鄰位置，改以職稱列、縣市、姓名與黨籍交叉辨識。
- 支援 `代理里長`、代理首長、議長／副議長、代表會主席等職稱。
- 依內政部頁面顯示的總筆數與每頁 200 筆，自動計算應抓頁數。
- 補入 `sms=11395` 模組參數，避免取得通用頁面框架而非人物清單。
- 同步報告新增解析診斷：首頁職稱列、詳細資訊標記、總筆數、抓取頁數與空白頁。
- 保留既有安全門檻；解析數量異常時不會用空資料覆蓋線上資料。
- 新增內政部頁面格式測試 fixture，GitHub Actions 每次每日同步前會先驗證。

## 排程

- 每天 03:00：現任公職與正式候選人。
- 每週日 04:20：中選會核心歷屆資料。
- 兩個 workflow 使用相同 concurrency group，撞期時後啟動者等待前一個完成。

## 上線方式

使用小更新檔覆蓋 repository 中相同路徑，透過 GitHub Desktop Commit、Pull、Push，完成後手動執行一次 `Daily official data sync`。
