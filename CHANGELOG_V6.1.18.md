# V6.1.18 UI 統整精簡版

## 介面層級

- 「政治版圖」改名為「現任公職」，地圖標題改為「現任公職地圖」。
- 縣市、鄉鎮市區與選舉區統一由地圖上方導覽，不再於左側重複顯示縣市。
- 移除畫面上的五格重複統計列，資料筆數與最後同步時間整合進結果標題。
- 版圖摘要集中至地區資訊面板，新增領先政黨與領先差距。

## 地圖與結果

- 第 11 屆立委選區官方範圍移入地區資訊，避免與地圖左下操作提示重疊。
- 現任公職顯示「政黨勢力／人數密度／首長黨籍」。
- 選舉結果只顯示「政黨版圖／席次密度」。
- 結果工具列保留搜尋與排序；展開、收合與人物比較收進「更多操作」。

## 次要資訊與手機版

- 歷屆人物、政黨與選區軌跡改成可展開區塊，只在選舉結果模式顯示。
- 官方資料健康與更新資訊改成可展開區塊。
- 手機版地圖高度限制在約 58% 視窗高度，地區導覽、結果工具列與收合區塊重新排版。
- 統一主要卡片的圓角、間距與陰影。

## 資料安全

- 本版為 UI 統整，不重新抓取或覆蓋現任與歷屆資料。
- 延續 V6.1.17 的分區載入、第 11 屆立委選區與官方資料唯讀原則。
