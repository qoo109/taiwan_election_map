# V6.1.6｜品牌統一、政黨分組與全資料同步

## 公開網站

- 瀏覽器標題、頁首左上角與頁尾統一改為「台灣選舉報報」。
- 公開介面不再顯示版本號；版本僅保留在專案內部更新紀錄。
- 現任公職移除「依資料層級／依姓名」排序選單，固定依政黨分組。
- 新增現任名單專用搜尋欄，可搜尋姓名、政黨、職位與地區。
- 每個政黨群組可個別展開／收合，並顯示該黨人數。
- 現任公職整區新增「收合現任公職」按鈕。
- 每個政黨初次只顯示 12 人，可按「再顯示」逐批載入，避免數千筆資料造成超長頁面與瀏覽器卡頓。

## 官方資料同步

- 手動更新預設為 `full` 現任公職範圍，包含議員、首長、代表、原住民區與村里長。
- 手動完整更新預設為歷屆 `all` 與年度 `all`。
- Daily workflow 最長執行時間由 60 分鐘提高到 120 分鐘。
- 內政部分頁加入重試、請求間隔與完整頁數抓取。
- 完整模式會抓取所有已設定地方公職來源；標準模式則省略大量村里長與代表頁面。
- 村里長最多頁數提高到 60 頁，鄉鎮市民代表提高到 20 頁。

## 安全原則

- 更新失敗時繼續保留上一版正式資料。
- 大量刪除保護與官方來源查核規則維持不變。
- 小更新檔不包含 `data/`，不會覆蓋 GitHub Actions 已同步的正式資料。
